# Быстрый запуск (Docker)

Проект полностью упакован в контейнеры. Не совсем понял как правильно собрать проект чтобы его отправить просто с Монго, поэтому сделал с докером, нужны только он 
и Монго.Компас, в корневой папке есть test.products.json чтобы посмотреть как оно выглядит с товарами сразу, сделана валидация email при регистрации, валидация 
заполненности полей при указании данных карты для оплаты, работает создание товаров (ссылку на картинку можно указать локальную или url из сети), личный кабинет не 
открывается без авторизации, в личном кабинете работает смена пароля. 

## Как запустить

1. переходим в me-react-shop:

В терминале проекта:
docker-compose up --build

Ссылки где что запускается:
Фронтенд: http://localhost:3000

Бэкенд (API): http://localhost:9001

MongoDB: mongodb://localhost:27017

База данных (Импорт товаров)
Чтобы на витрине появились тестовые товары, закиньте JSON в базу:
Подключитесь через MongoDB Compass к mongodb://localhost:27017.
Зайдите в базу test, коллекция products 
Нажмите Add Data, Import File и выберите файл test.products.json из корня проекта.
